Развертывание собственного телеграмм-бота для генерации изображений фрактального пламени.

Спсоб 1. С использованием беспладной интегрированной среды разработки IntelliJ IDEA (Рекомендуется).
1) Создайте новый Java проект с использованим любого из сборщиков (IntelliJ, Maven, Gradle) и openjdk-21 или новее.
2) Скопируйте содержание архива в корень проекта
3) Замените слово "token" в token.txt на значение токена своего телеграмм-бота.
4) Нажмите правой кнопкой мыши на файл bot.jar, затем левой на "Run 'bot.jar'"
5) Если все шаги были выполнены правильно - ваш телеграмм бот должен быть запущен.

Сбособ 2. Без использования сторонних IDE.
1) Переместите файлы из архива в отдельную папку
2) Замените слово "token" в token.txt на значение токена своего телеграмм-бота.
3) Придумайте кодовое слово для выключения телеграмм-бота и запишите его в exitWord.txt. Не оставляйте лишних символов, в том числе пробелов.
4) Запустите файл bot.jar, используя Java версии 21 или новее. Убедитесь в использовании корректной версии Java.
5) Ваш телеграмм-бот должен быть запущен. Чтобы остановить работу программы на компьютере, дважды отправьте телеграмм-боту кодовое слово из файла exitWord.txt. Убедитесь, что перед этим была введена команда /start.

* Создать своего телеграмм-бота и получить его токен можно по ссылке https://t.me/BotFather
** На компьютере должна быть установлена Java версии 21 или новее.
*** Стоит учитывать что моделирование изображений фрактального пламени требует значительных ресурсов компьютера.
